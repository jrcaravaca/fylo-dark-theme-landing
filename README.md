# Fylo Dark Theme Landing

## Description
A modern landing page for Fylo with a dark theme design. This is a React-based project showcasing a cloud storage service. Made for FrontEndMentor Challenge

## Features
- Responsive dark theme design
- Clean and modern UI
- Mobile-friendly layout
- Interactive components

## Installation
```bash
npm install
```

## Usage
```bash
npm start
```

## Build
```bash
npm run build
```

## Technologies
- React
- CSS3
- HTML5
